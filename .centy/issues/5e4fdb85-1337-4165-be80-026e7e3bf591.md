# Add docker-compose.yml for easy local development

No docker-compose.yml exists for running the project in Docker. This makes it harder for new contributors to get started.

## Requirements
- docker-compose.yml with the tmux-gateway service
- Mount .env file for configuration
- Expose HTTP and gRPC ports
- Add health check in compose using the /health endpoint
- Consider adding a grpcui sidecar container for gRPC exploration
- Document usage in README

## Optional Additions
- Add a Makefile target: make docker-up / make docker-down
- Add volume for persistent tmux sessions (if applicable)
