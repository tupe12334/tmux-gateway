# Add .env.example and CORS configuration

The project uses dotenvy for env vars but has no .env.example file, and no CORS is configured on the HTTP server.

## .env.example
Create a documented .env.example with all supported environment variables:
- HTTP_PORT=8080
- GRPC_PORT=50051
- RUST_LOG=tmux_gateway=info
- SHUTDOWN_TIMEOUT_SECS=30 (from graceful shutdown issue)

## CORS Configuration
- Add tower-http CORS middleware to the Axum router
- Support configurable allowed origins via CORS_ORIGINS env var
- Default to localhost origins for development
- Allow all standard methods and headers
- Add CORS_ORIGINS to .env.example
