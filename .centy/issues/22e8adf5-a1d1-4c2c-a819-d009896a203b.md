# Add LICENSE file to repository root

Cargo.toml declares license = "MIT" but there is no LICENSE file in the repository root.

## Why
- GitHub won't detect the license without the file
- Users can't verify the actual license terms
- Required for proper open-source compliance

## What to do
- Add a standard MIT LICENSE file to the repo root with the correct copyright holder and year
