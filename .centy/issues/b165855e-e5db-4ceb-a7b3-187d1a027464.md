# Add Makefile targets for common dev workflows

The Makefile only has a build target. Developers must remember raw cargo commands for testing, linting, formatting, and coverage.

## What to do
Add targets:
- make test — cargo test
- make lint — cargo clippy --workspace -- -D warnings
- make fmt — cargo fmt --all -- --check (and make fmt-fix for auto-fix)
- make check — cargo check --workspace
- make coverage — cargo tarpaulin (matching CI)
- make dev — cargo watch -x run (requires cargo-watch)
- make clean — cargo clean
- make help — list all targets with descriptions (default target)

## Acceptance criteria
- make help shows all available targets
- CI and local dev use the same commands via Makefile
