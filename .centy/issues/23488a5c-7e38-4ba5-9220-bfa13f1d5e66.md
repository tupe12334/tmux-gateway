# Add pane output accumulation buffer for time-windowed content collection

capture_pane provides point-in-time snapshots. For monitoring use cases, clients need output accumulated over a time window. Add an OutputBuffer domain concept that collects pane output over a configurable duration, returning all content produced in that window. Complements streaming (#8) for batch-oriented consumers.
