# Add domain-level pane output streaming abstraction

Issues #8 (WebSocket) and #97 (pipe-pane) handle streaming at transport and tmux levels respectively, but the domain has no streaming concept. Add an OutputStream trait and PaneOutputStream that domain operations can produce, enabling any transport layer to consume real-time pane output. Decouples streaming from transport protocol.
