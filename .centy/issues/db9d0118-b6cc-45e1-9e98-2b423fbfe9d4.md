# Add optional API key authentication

The server has no authentication. Any client can perform destructive operations like killing sessions.

## Implementation
- Add optional API key authentication via Authorization: Bearer header
- Environment variable: API_KEY (when set, auth is required; when empty, auth is disabled)
- Apply as Axum middleware for REST and GraphQL
- Apply as tonic interceptor for gRPC (via metadata)
- Return HTTP 401 Unauthorized / gRPC UNAUTHENTICATED when key is invalid
- Disable gRPC reflection when auth is enabled (or protect it with auth too)
- Document auth setup in README
