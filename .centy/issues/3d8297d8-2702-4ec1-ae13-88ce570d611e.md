# Add gRPC client SDK generation guide and buf configuration

The project exports .proto files and descriptors, but there is no guide for clients to generate language-specific gRPC stubs. No buf.yaml or buf.gen.yaml exists for structured proto management.

## What to do
- Add buf.yaml and buf.gen.yaml for proto linting and codegen
- Add a docs/grpc-clients.md with examples for generating clients in:
  - Python (grpcio-tools)
  - Node.js/TypeScript (grpc-tools or buf)
  - Go (protoc-gen-go-grpc)
  - Rust (tonic-build)
- Add a make proto-lint target using buf lint
- Add example client scripts in examples/ directory

## Acceptance criteria
- A developer can generate a working gRPC client in any major language using the documented steps
- buf lint passes on the proto file
