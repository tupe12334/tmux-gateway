# Add domain operation dry-run mode for previewing effects without execution

No way to preview what a domain operation would do before executing. Add a DryRunResult type and dry-run capability to destructive operations (kill-session, kill-window, split-window). Returns what would happen (sessions affected, panes removed, etc.) without actually executing. Essential for safety in production.
