# Make tmux install suggestion platform-aware in preflight checks

When tmux is not found, the preflight error message hardcodes \"install with: brew install tmux\" which is macOS-specific. Users on Linux or WSL see an unhelpful suggestion.

## What to do
- Detect the platform at runtime (cfg or uname)
- Suggest the correct package manager:
  - macOS: brew install tmux
  - Debian/Ubuntu: sudo apt install tmux
  - Fedora/RHEL: sudo dnf install tmux
  - Arch: sudo pacman -S tmux
  - Other: install tmux from https://github.com/tmux/tmux
- Add a note in README that tmux-gateway requires tmux on the host (Unix-only)

## Acceptance criteria
- Linux users see an appropriate install command
- README documents the tmux dependency clearly
