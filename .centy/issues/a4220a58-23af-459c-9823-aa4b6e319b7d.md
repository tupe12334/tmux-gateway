# Makefile clean target doesn't remove all generated schema files

## Problem
The Makefile `clean` target removes only `openapi.json` and `schema.graphql` from the schemas directory, but doesn't remove the generated `tmux_gateway.proto` and `tmux_gateway_descriptor.bin` files. Since the project uses code-first gRPC (proto is generated from Rust macros, not hand-maintained), all four schema files should be treated as generated artifacts.

## Current state
```makefile
clean:
	cargo clean
	rm -f schemas/openapi.json schemas/schema.graphql
```

Missing: `schemas/tmux_gateway.proto` and `schemas/tmux_gateway_descriptor.bin`.

## Proposed change
```makefile
clean:
	cargo clean
	rm -f schemas/openapi.json schemas/schema.graphql schemas/tmux_gateway.proto schemas/tmux_gateway_descriptor.bin
```

## Acceptance criteria
- [ ] `make clean` removes all four generated schema files
