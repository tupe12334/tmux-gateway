# Add synchronized multi-pane input domain operation

tmux synchronize-panes sends keystrokes to all panes in a window simultaneously. The domain has send_keys for single panes but no operation for synchronized multi-pane input. Add send_keys_synchronized(window_target, keys) and set_synchronize_panes(window_target, enabled). Essential for managing clusters of identical terminals.
