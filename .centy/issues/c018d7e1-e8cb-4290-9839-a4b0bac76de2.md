# Add session template inheritance with base and override layers

Issue #81 adds declarative session specs, but templates are flat with no inheritance. Add a SessionTemplate type supporting a base template plus override layers. Enables organization-wide defaults with per-project customizations. Templates compose via merge semantics (override replaces base values, arrays concatenate).
