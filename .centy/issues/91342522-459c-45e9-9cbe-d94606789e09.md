# CORS default origins incorrectly include gRPC port

In src/main.rs lines 50-55, the default CORS origins include both HTTP and gRPC ports:

````rust
let origins = env::var("CORS_ORIGINS").unwrap_or_else(|_| {
    format\!("http://localhost:{},http://localhost:{}", http_port, grpc_port)
});
````

## Problem:

* gRPC uses HTTP/2 and doesn’t need CORS
* http://localhost:50051 as a CORS origin doesn’t make sense - no browser app runs on the gRPC port
* The second origin should be a common frontend dev port (e.g., 3000, 5173) not the gRPC port

## Fix:

Change default to only include the HTTP port and common frontend dev ports:

````rust
format\!("http://localhost:{},http://localhost:3000", http_port)
````
